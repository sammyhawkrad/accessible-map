To navigate the map with the keyboard, use the following shortcuts. Left arrow, Move left. Right arrow, Move right. Up arrow, Move up. Down arrow, Move down. Plus key, Zoom in. Minus key, Zoom out. Hold Shift key and F key to search for a castle. When screen reader is active hold Shift key to pan the map with arrow keys.
