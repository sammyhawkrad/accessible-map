About this project

This project is developed as part of a master thesis in the International Master of Science in Cartography at TU Dresden. The master thesis topic is "Accessibility Approaches for Thematic Web Maps". The main objective of the thesis is to investigate how the Web Content Accessibility Guidelines (WCAG) can be applied to thematic web maps.

This project is a proof of concept of an accessible map based on the study results. The map shows castles in Austria and provides brief descriptions of each castle. The map is designed to be accessible to people with visual impairments and to be navigable with the keyboard.
